DataPredict is a machine and deep learning library for Roblox.

This project is created to address the lack of machine and deep learning libraries as well as the inflexibility of the available ones on Roblox.

It is also a way for me to apply my machine and deep learning knowledge into its physical form.

If there are any suggestions or issues for this library, don’t be afraid to reach out to me at my LinkedIn or in this DevForum thread.

In addition, if you are interested in Roblox tutorials and future projects by me, then you can subscribe to me in YouTube.
